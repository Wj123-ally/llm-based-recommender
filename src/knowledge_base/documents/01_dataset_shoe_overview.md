# 鞋类商品库导购范围

本知识库围绕当前鞋类商品数据补充导购解释。商品库以鞋类为主，常见鞋型包括皮鞋、板鞋、运动鞋、拖鞋、凉鞋、马丁靴、高跟鞋、靴子、帆布鞋、乐福鞋、妈妈鞋、跑步鞋、雨鞋、雪地靴、登山鞋和劳保鞋。

推荐时应优先结合商品标题和结构化字段，包括鞋型、颜色、材质、季节、人群、使用场景和功能。不要把非鞋类商品作为推荐重点；如果检索结果中出现箱包、清洁剂等噪声，应说明其不符合鞋类需求并优先选择鞋类商品。

当前商品常见功能词包括透气、防滑、舒适、轻便、防水、保暖、运动、显瘦。功能词通常来自标题和属性，适合作为推荐理由，但不能替代具体鞋型判断。例如“透气”可能出现在运动鞋、网面鞋、凉鞋和劳保鞋中，需要结合场景解释。

当前商品常见颜色包括白色、黑色、棕色、米色、拼色、灰色、红色和绿色。颜色可能来自文本或图片识别；当文本颜色和图片颜色冲突时，回答中应保守表述为“图片看起来偏某色”或“标题未明确颜色”。

当前商品常见季节包括夏季、春季、春秋、冬季和秋冬。夏季优先关注透气、凉感、防滑和轻便；冬季优先关注保暖、加绒、防水和防滑；春秋适合皮鞋、乐福鞋、板鞋、马丁靴和休闲运动鞋。

导购回答应先匹配用户的核心需求：鞋型、场景、性别/人群、颜色、季节、功能。再补充知识说明，例如为什么适合通勤、为什么适合宽脚、某种材质如何保养。
