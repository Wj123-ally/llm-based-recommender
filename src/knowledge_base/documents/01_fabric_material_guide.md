# 服装面料材质完全指南

## 概述

面料是服装的灵魂。了解常见面料的特性，能帮助你根据季节、场合、预算做出更好的购买决策。本指南涵盖天然纤维、化学纤维和混纺面料三大类。

---

## 天然纤维

### 棉 (Cotton)
- **优点**：吸湿透气、柔软亲肤、不易过敏、价格亲民
- **缺点**：易皱、缩水（首次洗涤约缩 3-5%）、弹性差
- **适用**：T 恤、衬衫、内衣、夏季服装
- **鉴别**：燃烧后呈灰白色粉末，有烧纸味

### 麻 (Linen)
- **优点**：透气性极佳、吸湿排汗、天然抗菌、越洗越软
- **缺点**：极易皱、手感粗糙、缺乏弹性
- **适用**：夏季衬衫、连衣裙、阔腿裤
- **鉴别**：燃烧特征与棉类似，纤维较粗硬

### 真丝 (Silk)
- **优点**：光泽华丽、触感丝滑、透气保湿、对皮肤摩擦小
- **缺点**：价格昂贵、不耐磨、怕汗渍、需干洗或手洗
- **适用**：连衣裙、衬衫、围巾、睡衣
- **等级**：桑蚕丝 > 柞蚕丝 > 绢丝。姆米数（MM）越高越厚实，常见 12-22MM

### 羊毛 (Wool)
- **优点**：保暖性好、弹性佳、吸湿不显潮、不易起静电
- **缺点**：可能扎人、易缩水（热水或机洗）、易虫蛀、价格较高
- **适用**：毛衣、大衣、西装、围巾
- **鉴别**：燃烧后有烧毛发气味，灰烬可捏碎
- **细分**：美利奴羊毛（细软不扎）、羊羔毛（初次剪毛最柔软）、马海毛（长而蓬松）

### 羊绒 (Cashmere)
- **优点**：极其柔软轻盈、保暖性是羊毛的 8 倍、不扎皮肤
- **缺点**：非常昂贵、强度低易破损、需精心保养
- **适用**：高端毛衣、围巾、大衣
- **鉴别**：纯羊绒燃烧后全部成灰；含化纤会结硬块

## 化学纤维

### 聚酯纤维 / 涤纶 (Polyester)
- **优点**：挺括不易皱、快干、耐磨、价格低
- **缺点**：透气性差、易起静电、易起球、吸湿性差
- **适用**：运动服、外套、填充棉
- **环保**：再生聚酯（rPET）使用回收塑料瓶制成

### 锦纶 / 尼龙 (Nylon)
- **优点**：弹性极好、耐磨、轻便、快干
- **缺点**：不耐日晒、透气差、易起静电
- **适用**：运动服、泳衣、丝袜、羽绒服面料

### 氨纶 / 莱卡 (Spandex)
- **优点**：超高弹性（可拉伸 5-8 倍）、回复性好
- **缺点**：不能单独使用、不耐氯漂
- **适用**：通常 2-5% 混纺于牛仔裤、紧身衣、运动服

### 粘胶纤维 / 人造丝 (Viscose/Rayon)
- **优点**：手感柔软顺滑、光泽好、透气优于涤纶
- **缺点**：湿强度差（湿水后易断裂）、易缩水、易皱
- **适用**：连衣裙、衬衫、里衬

### 莫代尔 (Modal)
- **优点**：比棉更软、吸湿性比棉高 50%、不易缩水
- **缺点**：价格高于普通粘胶
- **适用**：内衣、家居服、T 恤

### 莱赛尔 / 天丝 (Lyocell/Tencel)
- **优点**：环保工艺、强力好、手感如丝、可生物降解
- **缺点**：价格较高
- **适用**：高档衬衫、连衣裙、牛仔裤

## 常见混纺

| 混纺 | 特点 |
|------|------|
| 棉+涤纶 (CVC/TC) | 挺括不皱，适合衬衫工装 |
| 羊毛+涤纶 | 降低成本，改善抗皱 |
| 棉+氨纶 | 微弹，牛仔裤标配 |
| 真丝+棉 | 降成本，保光泽，可水洗 |
| 羊绒+羊毛 | 保柔软降价格 |

## 常见问题

**Q: "纯棉"和"全棉"有什么区别？**
A: 国家标准中，含棉量 ≥95% 可标纯棉，≥100% 标全棉。日常使用无实质差别。

**Q: 羊毛衫和羊绒衫怎么选？**
A: 贴身穿着选羊绒（不扎），外层保暖选羊毛（耐磨）。预算有限选美利奴羊毛，软度接近羊绒。

**Q: 夏季穿什么面料最凉快？**
A: 麻 > 真丝 > 棉 > 莫代尔 > 粘胶 > 涤纶。但麻易皱，真丝贵，棉是性价比之选。

## 关键要点

- 买贴身衣物优先天然纤维（棉、莫代尔、真丝）
- 运动服装选锦纶+氨纶混纺
- 冬季保暖选羊毛/羊绒，追求性价比选腈纶混纺
- 混纺不是坏事——合理混纺常能取长补短
- 看水洗标上的成分比例，不要只看商品标题
