# 常见鞋型选择指南

皮鞋适合通勤、商务、婚礼和正式场合。商品库中的皮鞋常见描述有真皮、牛皮、头层牛皮、软底、一脚蹬、正装、英伦风。推荐皮鞋时应关注鞋头、鞋底软硬、闭合方式和颜色。黑色皮鞋更正式，棕色皮鞋更休闲。

板鞋适合日常休闲、学生、街头和情侣搭配。板鞋通常外观稳定、搭配范围广，白色、黑色和拼色较常见。推荐板鞋时可以强调百搭、休闲、耐看，但如果用户要求运动性能，不要把普通板鞋当作专业跑步鞋。

运动鞋适合日常走路、轻运动、校园和休闲通勤。商品库中的运动鞋经常带有透气、轻便、舒适、老爹鞋、增高等描述。推荐时应区分“运动风休闲鞋”和“专业跑步鞋”。

拖鞋和凉鞋适合夏季、居家、浴室、沙滩和短距离外穿。拖鞋重点看防滑、软底、厚底、材质和是否适合浴室；凉鞋重点看鞋带稳定性、鞋底防滑、脚背包裹和是否磨脚。

马丁靴和靴子适合秋冬、机车、英伦、复古和户外风格。推荐时应关注鞋筒高度、鞋底防滑、是否加绒、是否防水、鞋面材质和重量。厚底靴显高但可能偏重。

高跟鞋适合正式场合、约会、婚礼和气质搭配。推荐时应关注跟高、跟型、鞋头、鞋面稳定性。粗跟通常比细跟更稳，低跟比高跟更适合长时间行走。

帆布鞋适合学生、休闲、春夏和轻便搭配。帆布透气但防水性弱，雨天不适合。白色帆布鞋百搭但不耐脏，黑色帆布鞋更耐看。

乐福鞋适合通勤、英伦风、轻商务和休闲正装。乐福鞋通常穿脱方便，适合春秋季；宽脚用户要关注鞋楦和前掌空间。
