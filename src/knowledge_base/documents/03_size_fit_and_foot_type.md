# 鞋码、脚型与舒适度

鞋码选择应以脚长、脚宽和穿着场景为基础。只看平时码数不够准确，因为不同鞋型、品牌和鞋楦会有偏差。运动鞋、跑鞋和冬季加绒鞋通常需要预留更多空间；皮鞋和乐福鞋如果过大容易掉跟，过小容易压脚背和挤脚趾。

大码女鞋常见于 40、41、42、43 码。推荐大码鞋时，应优先关注标题中是否明确“大码”“41-43”“宽脚”“胖脚”，以及鞋头是否方头、圆头或软面。尖头鞋即使有大码，也可能不适合宽脚。

宽脚用户适合圆头、方头、软皮、软底、网面、鞋面有弹性或鞋带可调的款式。应谨慎选择尖头、窄楦、硬皮和一字带过细的鞋。

高脚背用户适合鞋带、魔术贴、弹力鞋面或开口较大的鞋。浅口单鞋、硬面乐福鞋和窄口马丁靴可能压脚背。

老人鞋和妈妈鞋应优先考虑软底、防滑、轻便、穿脱方便和稳定支撑。不要只强调显高或厚底，厚底过高可能影响稳定性。

童鞋应关注轻便、防滑、透气、脚趾活动空间和鞋底弯折性。儿童运动凉鞋和沙滩鞋适合夏季户外，但浴室场景仍要看防滑鞋底。

试穿判断可以参考：脚趾前方有少量余量，前掌不挤，脚跟不明显滑动，走路时鞋面不压脚背，鞋底弯折位置接近脚掌自然弯折位置。
