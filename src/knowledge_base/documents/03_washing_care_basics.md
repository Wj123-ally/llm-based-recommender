# 服装洗涤保养基础指南

## 概述

80% 的衣服不是穿坏的，是洗坏的。读懂水洗标、掌握正确的洗涤方法，能让一件衣服的寿命延长 2-3 倍。

---

## 水洗标符号解读

国际通用的洗涤符号分为五大类：

| 符号类型 | 含义 | 关键信息 |
|----------|------|----------|
| 水盆 | 水洗 | 盆中数字 = 最高水温（30/40/60/95°C），下方横线 = 缓和程序 |
| 三角 | 漂白 | 空三角 = 可漂白，三角带× = 不可漂白 |
| 方块 | 干燥 | 方块中竖线 = 悬挂晾干，横线 = 平铺晾干，圆点 = 烘干温度 |
| 熨斗 | 熨烫 | 圆点数量 = 温度档位（1低温/2中温/3高温），带× = 不可熨 |
| 圆环 | 干洗 | P/F = 不同干洗溶剂，带× = 不可干洗 |

**最常用组合速记**：
- 🧺 30°C + 下方一横 = 手洗模式或机洗轻柔档，30°C 以下水温
- 🧺 手形符号 = 必须手洗，不可机洗
- ⬜ + × = 禁止烘干（羊毛、真丝常见）

---

## 水温选择

| 水温 | 适用 | 不适合 |
|------|------|--------|
| **冷水** (<30°C) | 深色、易缩水、精细面料 | 重度油污 |
| **温水** (30-40°C) | 绝大多数日常衣物 | 羊毛、真丝 |
| **热水** (>50°C) | 白色棉质、毛巾床品（杀菌） | 深色、化纤、弹性面料 |

**黄金法则**：不确定时一律用冷水。热水是多数缩水、褪色、变形的元凶。

---

## 洗涤剂选择

| 类型 | 适用 | 特点 |
|------|------|------|
| 普通洗衣液 | 日常棉质、化纤 | 中性温和 |
| 丝毛净 | 羊毛、真丝 | 中性 pH，含柔顺成分 |
| 羽绒专用 | 羽绒服 | 不破坏羽绒油脂 |
| 彩漂液 | 彩色衣物 | 含氧漂白不伤色 |
| 氯漂 | 白色棉质 | 强力漂白，伤纤维 |
| 皂粉 | 棉质内衣 | 天然成分，去污力一般 |

**避雷**：
- 羊毛/真丝不能用普通洗衣液（碱性破坏蛋白质纤维）
- 黑色衣物慎用含荧光增白剂的洗涤剂
- 柔顺剂不要用在运动服上（堵塞透气微孔）

---

## 机洗 vs 手洗

**必须手洗**：真丝、羊绒、蕾丝、有亮片/钉珠装饰、内衣（延长寿命）

**机洗安全**：棉、涤纶、锦纶、牛仔裤、运动服

**机洗保护技巧**：
1. 用洗衣袋装精细衣物
2. 拉链拉好、扣子解开
3. 深浅色分开
4. 不超过洗衣机容量 3/4
5. 选"轻柔"或"羊毛"程序（脱水转速更低）

---

## 晾晒指南

| 衣物类型 | 晾晒方式 | 原因 |
|----------|----------|------|
| 羊毛衫 | **平铺**晾干 | 悬挂会因自重拉长变形 |
| 真丝 | 阴凉处晾干 | 日晒会导致褪色和纤维劣化 |
| T 恤 | 可悬挂 | 棉质不易变形 |
| 牛仔裤 | 翻面晾晒 | 减少褪色 |
| 羽绒服 | 低温烘干 + 网球 | 防结块，恢复蓬松 |
| 深色衣物 | 翻面阴干 | 防止日晒褪色 |

## 常见问题

**Q: 新衣服要不要先洗再穿？**
A: 建议先洗。新衣服上可能有生产过程中的化学残留（甲醛、染料固色剂），贴身穿的更要注意。

**Q: 衣服越洗越硬怎么办？**
A: 可能是洗涤剂残留或硬水矿物质堆积。试一下：白醋当柔顺剂（半杯），或者减少洗涤剂用量。

**Q: 可以用沐浴露/洗洁精洗衣服吗？**
A: 紧急情况可以，但不推荐。它们的配方不是为衣物纤维设计的，长期使用可能损伤面料。

## 关键要点

- 养成看水洗标的习惯——每件衣服的洗涤要求都写在标签上
- 冷水 + 轻柔模式 = 绝大多数衣物的安全选择
- 羊毛和真丝的敌人是热水、强碱、机械力
- 平铺晾干是羊毛制品的唯一正确晾法
- 少用洗涤剂比多用好——残留的洗涤剂会吸附灰尘让衣服变黄
