# 功能词与使用场景

透气适合夏季、运动、跑步和长时间穿着。网面运动鞋、凉鞋、帆布鞋通常更容易满足透气需求。真皮鞋也可能有一定透气性，但不能和网面鞋等同。

防滑适合浴室拖鞋、雨鞋、户外鞋、老人鞋、劳保鞋和雨天通勤。推荐防滑鞋时应关注鞋底纹路、橡胶底、软底和使用场景。浴室防滑和户外防滑不是同一需求，浴室更关注湿滑地面，户外更关注泥地、石路和坡面。

防水适合雨鞋、户外鞋、雪地靴和工作鞋。普通网面、帆布和皮鞋即使标题有防滑，也不代表防水。真皮鞋遇水后需要及时擦干和护理。

保暖适合冬季、秋冬、雪地靴、加绒靴、棉鞋和冬季运动鞋。保暖款通常会牺牲一部分透气性，用户如果要长时间室内穿，应避免过厚。

轻便适合通勤、旅行、老人鞋、妈妈鞋、跑步和夏季穿着。轻便不等于支撑好，专业跑步或长距离行走还要看缓震和足弓支撑。

舒适是综合结果，来自鞋楦、鞋底、鞋面、鞋垫、重量和尺码。商品标题里的“舒适”只能作为参考，推荐时应结合软底、宽脚、大码、网面或真皮等具体信息解释。

显瘦常见于女鞋、高跟鞋、尖头鞋、厚底鞋和部分靴子。显瘦类商品要同时关注舒适度，尖头和高跟可能不适合长时间步行。

通勤场景适合皮鞋、乐福鞋、休闲运动鞋、板鞋和低跟单鞋。跑步场景优先跑步鞋和运动鞋。居家场景优先拖鞋、凉拖、软底和防滑。户外场景优先防滑、防水、耐磨和包裹性。
