# 服装尺码选购完全指南

## 概述

网购最怕的就是尺码不对。了解自己的身体数据、看懂尺码表、掌握各品牌的尺码偏差，是减少退换货率的关键。

---

## 身体测量方法

### 必备工具
- 软尺（裁缝尺），不要用卷尺（太硬不准）
- 测量时穿贴身薄衣，自然站立

### 上装关键数据

| 测量部位 | 方法 | 用途 |
|----------|------|------|
| **肩宽** | 从左肩端点 → 后颈点 → 右肩端点 | 外套、衬衫合身度 |
| **胸围** | 绕胸部最丰满处水平一圈，正常呼吸 | 上衣、连衣裙 |
| **腰围** | 肚脐上方约 2cm 最细处 | 高腰裤、收腰连衣裙 |
| **衣长** | 侧颈点（肩与颈交界）到想要的长度 | 确认款式长度 |
| **袖长** | 肩端点沿手臂外侧到手腕 | 长袖上衣、外套 |

### 下装关键数据

| 测量部位 | 方法 | 用途 |
|----------|------|------|
| **腰围** | 自然腰线处水平一圈（不是低腰位置） | 裤腰 |
| **臀围** | 臀部最丰满处水平一圈 | 裤子、半身裙 |
| **大腿围** | 大腿根部最粗处一圈 | 紧身裤、短裤 |
| **裤长** | 腰头到脚踝/地面 | 确认裤长类型 |
| **档长/前档** | 前腰头到裆底 | 高腰/低腰判断 |

### 鞋码

中国鞋码 = 脚长（cm），如 230 = 脚长 23cm = 36 码。测量时下午或傍晚（脚会略微肿胀），双脚都量取较大值。

## 尺码表解读

### 常见尺码对照（女装）

| 尺码 | S | M | L | XL | 2XL |
|------|---|---|---|----|-----|
| 胸围(cm) | 80-84 | 84-88 | 88-92 | 92-96 | 96-102 |
| 腰围(cm) | 64-68 | 68-72 | 72-76 | 76-80 | 80-86 |
| 臀围(cm) | 86-90 | 90-94 | 94-98 | 98-102 | 102-108 |

**重要**：以上仅为参考。韩国品牌偏小 1-2 码，欧美品牌偏大 1-2 码，日系标准与中国接近但更瘦。

## 网购选码策略

### 1. 看模特数据
商品详情页如果有模特试穿信息（身高/体重/三围/穿几码），这是最准的参考。找一个和自己身材接近的模特，照她的码买。

### 2. 看买家秀
看和自己身材相似的买家评价，特别是带图的。注意看"偏大"、"偏小"、"刚好"的出现频率。

### 3. 了解品牌尺码偏好
- ZARA / H&M: 欧码偏大，通常买小一码
- 优衣库: 标准日码，按尺码表偏准
- 韩系品牌（Chuu, Stylenanda）: 版型偏小，买大一码
- 国产品牌（伊芙丽、太平鸟等）: 相对标准

### 4. 退换货策略
买两码，留下合适的退掉另一件。虽然麻烦，但比买一件不合身再换更省时间。

## 特殊体型选购建议

| 体型特点 | 选购策略 |
|----------|----------|
| 肩宽 | 避免泡泡袖、垫肩；选插肩袖、V 领拉长颈部视觉 |
| 胸大 | 选深 V/U 领，避免高领和小圆领；注意胸围尺码 |
| 腰粗/苹果型 | 选 A 字版型、帝国腰线（胸下收褶）；避免紧身收腰 |
| 胯宽/梨型 | 选阔腿裤、A 字裙；避免紧身包臀 |
| 小个子(<158cm) | 高腰线是第一要义；九分裤=你的全长裤；避免超长款 |
| 大个子(>172cm) | 确认袖长和裤长是否够；欧美品牌更友好 |

## 常见问题

**Q: 网购裤子总是不合身怎么办？**
A: 记住自己的三个数据：腰围、臀围、大腿围。买裤子时对照商品尺码表比较这三个数（而不是只凭 S/M/L）。裤长可以改，腰和臀改不了。

**Q: 鞋子买大了一码，有救吗？**
A: 加一个全掌鞋垫 + 后跟贴，可以弥补半码到一码。买小了基本无解。

**Q: 儿童衣服怎么选码？**
A: 按身高（cm）选，不是年龄。国内童装标准：身高 80=1 岁左右，100=3 岁，110=4-5 岁，120=6-7 岁，130=8-9 岁。

## 关键要点

- 先量体，再购物——花 5 分钟量好自己的三围、肩宽、脚长
- S/M/L 不可靠——同一品牌不同款式都可能差一个码
- 软尺是网购必备工具，挂衣架旁随用随取
- 买两码退一码，是一个很实用的策略
- 裤长可改，腰和臀改不了——所以买裤子优先看腰臀数据
