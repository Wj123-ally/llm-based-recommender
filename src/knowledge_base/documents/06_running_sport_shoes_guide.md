# 运动鞋、跑步鞋与户外鞋

运动鞋是宽泛概念，商品库中既有休闲运动鞋、老爹鞋，也有跑步鞋、训练鞋、篮球鞋和户外鞋。推荐时应根据用户场景区分，不要把所有运动鞋都当作专业运动装备。

跑步鞋应关注缓震、轻便、透气、鞋底抓地和脚跟稳定。短距离慢跑和日常走路可以选择轻便透气款；长距离跑步需要更明确的缓震和支撑信息。如果商品只写“运动鞋”“老爹鞋”，通常更适合日常运动风，而不是严肃跑步训练。

训练鞋适合健身房、综合训练和力量训练，通常需要稳定支撑和防滑鞋底。它不一定适合长距离跑步。

篮球鞋需要脚踝支撑、缓震和抓地。商品库中篮球鞋数量很少，如果用户明确要篮球鞋，应优先检索标题中包含篮球鞋的商品，不要用普通板鞋替代。

户外鞋、登山鞋和徒步鞋应关注防滑、防水、耐磨、包裹性和鞋底硬度。轻户外和城市通勤可以接受休闲户外鞋；山地徒步需要更专业的鞋底和支撑。

老爹鞋通常偏休闲和增高，适合日常搭配。厚底老爹鞋显高但可能偏重，不适合对轻量跑步有要求的用户。

运动鞋颜色选择上，白色百搭但不耐脏，黑色耐脏且适合通勤，拼色更有潮流感但搭配限制更高。
