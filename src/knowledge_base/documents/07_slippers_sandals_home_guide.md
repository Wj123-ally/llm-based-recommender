# 拖鞋、凉鞋与居家场景

拖鞋适合居家、浴室、夏季和短距离外穿。商品库中拖鞋常见卖点包括厚底、软底、防滑、防臭、浴室、洗澡、情侣和凉拖。推荐拖鞋时应先判断用户是居家穿、浴室穿还是外穿。

浴室拖鞋应优先考虑防滑、排水、易清洁和鞋底抓地。厚底拖鞋脚感好，但如果鞋底纹路不足，湿滑地面仍可能不安全。

居家拖鞋应关注软底、轻便、脚感和防臭。夏季居家适合凉拖、EVA 或轻便软底；冬季居家适合棉拖或加绒款。

外穿凉拖和半拖应关注鞋面稳定性、鞋底厚度、脚掌承托和是否容易掉跟。只靠细带固定的凉拖更适合短距离或轻场景。

凉鞋适合夏季、通勤、度假和日常外穿。平底凉鞋更舒适，粗跟凉鞋更稳，细跟凉鞋更适合短时间正式或约会场景。

儿童凉鞋和沙滩鞋应关注防滑、包头或脚趾保护、轻便和固定方式。魔术贴通常比单纯套脚更便于调节。

拖鞋和凉鞋的颜色以米色、黑色、白色、棕色常见。米色和白色更清爽，黑色更耐脏，棕色更自然休闲。
