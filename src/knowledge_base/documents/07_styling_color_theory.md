# 服装色彩搭配原理

## 概述

色彩是穿搭的第一语言。掌握基本的色彩搭配法则，可以让最简单的单品组合出高级感。

---

## 色轮基础

12 色环是最实用的搭配工具：

```
        红
    紫红   橙红
  紫          橙
    蓝紫   黄橙
  蓝          黄
    蓝绿   黄绿
        绿
```

### 色彩三属性
- **色相 (Hue)**：红/橙/黄/绿/蓝/紫
- **明度 (Value)**：浅色→深色（加白→加黑）
- **饱和度 (Saturation)**：鲜艳→灰暗

---

## 六大搭配法则

### 1. 同色系搭配（最安全）
同一种颜色的不同明度和饱和度组合。如：浅驼 + 焦糖 + 深棕。

**效果**：高级、统一、拉长身形
**适用**：职场、正式场合、想显高时

### 2. 邻近色搭配（最和谐）
色轮上相邻 2-3 个色。如：蓝 + 绿、橙 + 黄、紫红 + 蓝紫。

**效果**：和谐但不单调，比同色系更有层次
**适用**：日常通勤、休闲

### 3. 互补色搭配（最大胆）
色轮上相对的两个色。如：蓝 + 橙、红 + 绿、紫 + 黄。

**效果**：视觉冲击力强，抓眼球
**适用**：休闲场合、想被注意时
**注意**：降低一方饱和度（如墨绿+暗红）比两方都鲜艳更高级

### 4. 分裂互补色（互补但柔和）
选一个主打色，配其互补色两侧的邻近色。如：主打蓝，配橙红和橙黄。

**效果**：有对比但不刺眼
**适用**：想要对比但不会太扎眼的场合

### 5. 三角配色（三色平衡）
色轮上等距的三个色。如：红 + 黄 + 蓝。

**效果**：活泼、平衡
**适用**：休闲、度假、运动

### 6. 中性色万能法
黑、白、灰、米、驼、藏蓝、军绿。任何鲜艳色 + 中性色 = 安全好看。

**公式**：1 件彩色单品 + 其余中性色 = 基础搭配公式

## 肤色与色彩选择

| 肤色 | 推荐色 | 避雷色 |
|------|--------|--------|
| **冷白皮** | 几乎所有色，尤其粉、紫、蓝 | 几乎没有 |
| **暖黄皮** | 驼色、砖红、墨绿、姜黄、酒红 | 荧光色、纯白（选米白）、紫色 |
| **深色皮** | 亮色大胆穿！明黄、橙色、宝蓝、白 | 不明显的暗色（藏蓝、深棕避雷） |
| **中性皮** | 大部分暖调和中性调都可以 | 看具体明度 |

**一句话**：不确定自己肤色时，驼色和藏蓝是所有人的安全色。

---

## 场合与配色

| 场合 | 推荐配色 | 避免 |
|------|----------|------|
| **面试** | 黑/藏蓝/灰 + 白/浅蓝 | 大红大绿、大面积图案 |
| **通勤** | 米+白+驼、藏蓝+白、灰+粉 | 荧光色、豹纹 |
| **约会** | 酒红+米白、粉色+白、黑+红 | 看对方偏好 |
| **婚礼嘉宾** | 柔和色系（莫兰迪色）、粉色、蓝色 | 纯白（新娘专属）、全黑（国内部分地区忌讳） |
| **度假** | 大胆彩色！姜黄、亮蓝、橘、小白裙 | 没有限制 |

---

## 印花搭配原则

| 规则 | 说明 |
|------|------|
| **一花一素** | 印花上衣配纯色下装，或反过来 |
| **同底色法则** | 印花的主色中选一个作为其他单品的颜色 |
| **大小对比** | 大印花 + 小配饰（小包、细链），小碎花 + 大配饰（大包包） |
| **条纹 + 纯色** | 条纹是"伪纯色"，当成中性印花处理 |

## 常见问题

**Q: 穿一身黑真的可以百搭吗？**
A: 可以但要注意面料对比——黑毛衣 + 黑皮裤 + 黑皮鞋 = 有层次的全黑。如果全是同一种面料质感会像修女。

**Q: 红配绿真的很土吗？**
A: 不是！关键是明度和饱和度——暗红 + 墨绿是高级的秋冬配色（圣诞色来源），大红 + 正绿确实难驾驭。降低一方饱和度就对了。

**Q: 不会配色怎么办？最简单的入门法？**
A: "中性色做底色，一种彩色做亮点"。比如全身米白 + 一个红色包包，永远不会出错。

## 关键要点

- 新手从同色系开始，熟练后玩邻近色，大师才碰互补色
- 中性色（黑白灰米驼藏蓝）是衣橱的默默奉献者
- 肤色是你选颜色的第一参考系
- 降低饱和度 = 增加高级感（莫兰迪色的秘密）
- 不确定穿什么颜色时，驼色 + 白色永远是你的朋友
