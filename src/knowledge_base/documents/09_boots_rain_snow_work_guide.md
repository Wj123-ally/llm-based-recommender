# 靴子、雨鞋、雪地靴与工作鞋

马丁靴适合秋冬、机车、英伦、复古和街头风格。商品库中常见厚底、中筒、真皮、切尔西、英伦风等描述。厚底马丁靴显高，但可能偏重；真皮马丁靴更耐看，但需要磨合。

短靴适合通勤和日常搭配，中筒靴更有风格但对小腿围和腿型要求更高。推荐靴子时应关注筒高、跟高、鞋头、鞋底防滑和是否加绒。

雪地靴适合冬季保暖。加绒、羊毛、棉鞋、面包鞋等词通常与保暖相关。雪地靴不一定防水，雨雪天气还要看鞋面和鞋底描述。

雨鞋和雨靴适合雨天、户外、清洁和涉水场景。推荐时应关注防水、防滑、筒高和重量。轻便雨鞋适合日常雨天，长筒雨靴适合更强涉水需求。

劳保鞋和工作鞋适合工地、电焊、防砸、防刺穿等场景。商品库中劳保鞋可能带有钢头、防砸、防刺穿、软底、透气、轻便等描述。推荐时必须优先满足安全功能，不能只看外观。

户外靴、登山鞋和徒步鞋应关注鞋底抓地、防水、耐磨和包裹性。城市户外和专业登山需求不同，商品标题没有明确专业功能时应保守推荐。

冬季靴类常见黑色、棕色、米色。黑色耐脏且百搭，棕色适合复古和户外风，米色更柔和但清洁压力更高。
