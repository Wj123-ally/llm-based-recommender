# 配饰搭配法则

## 概述

配饰是穿搭中的"句号"——没有它，再好的衣服也显得未完成。包包、鞋子、首饰、围巾，用好配饰可以让基础款瞬间升级。

---

## 包包搭配

### 按体型选包

| 体型 | 适合包型 | 避免包型 |
|------|----------|----------|
| 小个子(<158cm) | 小号/中号、短肩带、斜挎 | 超大 tote、过长的链条包（压身高） |
| 高个子(>170cm) | 中号/大号、长肩带 | 过小的 mini 包（比例失调） |
| 丰满 | 中型、线条简洁 | 过小（显胖）或过大（显壮） |
| 瘦小 | 各种大小，oversized 也没问题 | 几乎没有 |

### 按场合选包

| 场合 | 包型 |
|------|------|
| 通勤 | 托特包/Tote、中号手提包 |
| 约会 | 链条包、马鞍包、小号斜挎 |
| 晚宴 | 手拿包/Clutch、mini 链条 |
| 旅行 | 双肩包、大号 tote |
| 逛街 | 斜挎包（解放双手） |

### 包与鞋的搭配
包和鞋最好是同色系或同材质——这是最简单的高级感公式。如棕色皮包 + 棕色皮鞋 = 协调。

---

## 鞋子搭配

### 基础鞋型

| 鞋型 | 搭配建议 | 万能度 |
|------|----------|--------|
| 小白鞋 | 配一切（除了正式西装和晚礼服） | ★★★★★ |
| 尖头平底 | 配九分裤、裙装、通勤 | ★★★★★ |
| 乐福鞋 | 配西装裤、牛仔裤 | ★★★★ |
| 小猫跟(3-5cm) | 配裙装、通勤 | ★★★★ |
| 一字带凉鞋 | 配连衣裙、阔腿裤 | ★★★★ |
| 切尔西靴 | 配紧身裤、牛仔裤 | ★★★ |

### 鞋的颜色
- 裸色/米色鞋 = 视觉延长腿部线条，最显腿长
- 黑色 = 百搭
- 红色/亮色 = 全身的亮点，其余保持素色

---

## 首饰搭配

### 分层法则

**项链**：
- 锁骨链(35-40cm) + 中长链(45-50cm) + 长链(55-70cm) = 三层叠戴
- 原则：粗细对比、长短错落
- V 领配吊坠项链，圆领配长链，一字领配 choker 或锁骨链

**耳环**：
- 圆脸 → 长条/流线型（拉长脸型）
- 长脸 → 圆形/短款（加宽视觉）
- 方脸 → 圆润/曲线设计（柔化棱角）
- 心形脸 → 耳钉/小圈都可以

**手链/手表**：
- 手表 + 细手链叠戴 = 经典组合
- 一只手戴 1-2 条即可，不必两只手都戴
- 金属色统一：金表配金链，银表配银链

### 亮点规则
全身配饰亮点不超过 2 个。如果你戴了大耳环 + 粗项链 + 叠戴手镯 + 大戒指 = 过载。选 2 处重点，其余简约。

---

## 围巾 / 丝巾

围巾是最被低估的配饰——它可以是围巾、发带、包带、腰带、手腕装饰。

**按季节**：
- 春秋：真丝方巾（90×90cm） = 经典百搭
- 冬季：羊绒/羊毛长巾 = 保暖 + 质感

**简单的丝巾用法**：
1. 绕颈一周打个结（法式 effortless）
2. 当发带绑马尾
3. 系在包带上（立刻让旧包变新）
4. 当手链绕几圈

## 常见问题

**Q: 配饰的颜色要和衣服一样吗？**
A: 不需要完全一样。同色系或互补色都可以。米色衣服 + 金色首饰、黑色衣服 + 银色首饰是基本公式。

**Q: 投资一个好包还是多买几个便宜的？**
A: 一只中性色（黑/驼/灰）的好质感包 + 2-3 个平价彩色/季节款 > 5 个廉价包。一只好包可以用 5-10 年。

**Q: 金银首饰可以混戴吗？**
A: 可以！但要注意有意识地混。一条金银双色的手链作为"桥梁"连接金戒指和银耳环。这叫做"故意混搭"而非"没注意到"。

## 关键要点

- 配饰不在多，在"对"—— 2 个亮点 > 5 个亮点
- 包和鞋的颜色一致是最简单的品质感公式
- 耳环的形状应该和脸型互补
- 一条丝巾 10 种用法，是性价比最高的配饰
