# 鞋类清洁、保养与收纳

真皮鞋应避免长时间泡水。日常用软布擦灰，必要时使用皮革护理膏或鞋油。淋雨后先擦干表面，再放在阴凉通风处自然干燥，不要暴晒或用高温烘干。

牛皮和头层牛皮鞋适合定期护理。使用鞋撑可以维持鞋型，减少折痕。浅色皮鞋更容易染色，穿着时应避免与深色下装或易掉色材质长时间摩擦。

网面运动鞋可以用软刷、温和清洁剂和湿布清洁。不要大力刷洗网面，避免变形或起毛。清洗后应阴干，避免太阳直晒导致胶水老化。

帆布鞋可局部刷洗，但白色帆布容易泛黄。清洁后可用纸巾包裹表面再阴干，减少水痕和黄渍。

麂皮和绒面鞋不能直接水洗。应使用麂皮刷轻刷灰尘，污渍严重时使用专用清洁剂。雨天不建议穿麂皮鞋。

拖鞋和凉鞋通常更易清洁。EVA、橡胶和塑料材质可用清水和软刷处理；带皮面、织物或装饰的凉鞋不要长时间浸泡。

加绒鞋和雪地靴要注意除湿和通风。穿后可以放入干燥剂，避免异味和霉变。内里潮湿时不要密封收纳。

除味可以使用通风、干燥剂、鞋用除味包或更换鞋垫。不要用浓烈香味掩盖潮湿异味，根本问题通常是汗液和湿气。

长期收纳前应先清洁、晾干，再放入鞋盒或防尘袋。靴子可使用靴撑或填充物保持筒形。
