# 衣物修复与保养

## 概述

衣服出了问题不一定就要扔。起球、缩水、褪色、拉链卡住——大多数常见问题都有 DIY 修复方法。

---

## 起毛起球

### 原因
任何短纤维面料都会起球（棉、羊毛、羊绒尤其常见）。这不是质量问题，是物理特性。

### 解决
1. **毛球修剪器**（¥20-50）：最有效的工具，轻轻划过面料表面即可
2. **剃须刀**：没有修剪器时的替代方案，注意力度
3. **不要用手拔**——会扯出更多纤维，加速起球
4. **预防**：洗涤时翻面、用洗衣袋、减少摩擦

---

## 缩水恢复

### 羊毛/羊绒缩水
1. 温水（30°C 左右）+ 适量护发素（或婴儿洗发水），浸泡 30 分钟
2. 护发素软化羊毛鳞片，使其可被重新拉开
3. 轻轻挤压水分（不要拧），平铺在大毛巾上
4. 边拉伸边塑形至原尺寸，平铺晾干
5. **成功率**：轻度缩水 70-80% 恢复，严重毡化基本不可逆

### 棉质缩水
棉的缩水通常是可逆的——再洗一次，在微湿状态下用力拉伸然后自然晾干。

## 褪色与染色

### 白衣发黄
- **氧系漂白**（彩漂液/过碳酸钠）：温水浸泡 1-2 小时，对白色尤有效
- **白醋 + 小苏打**：天然方法，效果温和
- **日晒**：洗净后太阳下晾晒（紫外线有天然漂白作用，但真丝/羊毛不行）
- 氯漂（84 消毒液）只在万不得已时用——会损伤纤维

### 深色衣物褪色
- 首次洗涤加盐或白醋浸泡固色（效果有限但不是谣言）
- 翻面洗 + 冷水 + 阴干 = 最有效的防褪色方法
- 褪色严重时可用专用染色剂（超市有售）重新染色

### 串色急救
白色衣物被染色：氧系漂白剂 + 热水浸泡
彩色衣物被染色：防染色布/吸色片（洗衣机里丢一片，吸附游离染料）

## 破洞与开线

### 小破洞
- **纺织胶水/布料胶**：1cm 以下小洞直接粘合
- **补丁贴**：烫上去的补丁（¥5-10 一包），选同色或故意选对比色当装饰
- **织补**：精细衣物（羊绒、真丝）建议找专业织补师傅

### 纽扣掉落
- 备用扣通常缝在水洗标旁边——检查一下
- 缝扣子诀窍：缠线圈（在扣子和面料之间留 2-3mm 缠线柱），这样扣子更牢固

### 拉链问题
| 问题 | 解决 |
|------|------|
| 拉链卡住 | 蜡烛/肥皂/铅笔芯在拉链齿上来回摩擦润滑 |
| 拉链拉不拢 | 用钳子轻轻夹紧拉链头两侧（各 0.5mm） |
| 拉链脱轨 | 拉链头剪掉一小段齿，重新穿入拉头，加一个止挡 |
| 拉链坏了 | 街边修鞋摊 10 元换新拉链头 |

## 去污急救

| 污渍 | 急救方法 |
|------|----------|
| 咖啡/茶 | 立刻冷水冲，不要用热水（定型色素） |
| 红酒 | 撒盐吸收 → 冷水冲洗 → 氧漂 |
| 油渍 | 洗洁精直接涂抹 + 轻搓（洗洁精专克油） |
| 口红 | 酒精/卸妆油擦拭，再洗涤 |
| 汗渍 | 白醋 + 水浸泡，不要用热水（凝固蛋白质） |
| 血渍 | 立刻冷水浸泡 + 肥皂搓（热水会凝固血液蛋白） |
| 墨水 | 酒精/牛奶浸泡 30 分钟 |
| 口香糖 | 冰箱冷冻 → 硬了剥掉 |

**终极急救原则**：冷水 > 热水，立刻处理 > 等待。

## 收纳保存

| 衣物类型 | 收纳方式 |
|----------|----------|
| 羊毛衫 | 折叠平放（悬挂会变形） |
| 皮衣 | 宽肩衣架悬挂，防尘罩透气不密封 |
| 羽绒服 | 悬挂 + 透气防尘罩（不要压缩） |
| 真丝 | 折叠 + 防潮纸 |
| 西装/大衣 | 好衣架（宽肩弧形）+ 透气防尘罩 |
| 鞋子 | 鞋撑保持形状（或塞报纸） |

**防虫防蛀**：
- 樟脑丸/防蛀片（化学）或薰衣草/雪松木（天然）
- 收起来前一定要洗干净（汗渍和食物残渣最招虫）
- 换季时晾晒（不可暴晒）

## 常见问题

**Q: 衣服起球了到底是不是质量问题？**
A: 不一定是。天然纤维（尤其是羊毛和羊绒）起球是纤维短的自然结果。100% 化纤反而更不容易起球。国际大牌的羊绒衫也起球。

**Q: 破了的衣服还值得修吗？**
A: 看两个因素：衣物的情感价值（特别喜欢/有纪念意义）和重置成本（贵的修、便宜的扔）。一般的拉链/纽扣/小洞修复 10-50 元就能搞定。

## 关键要点

- 毛球修剪器是衣橱必备小工具
- 缩水的羊毛衫 + 护发素浸泡 = 有机会救回来
- 任何污渍都应在干燥前立刻用冷水处理
- 防虫的关键是洗净再收纳
- 一件好衣服修三次比扔了买新的更划算
