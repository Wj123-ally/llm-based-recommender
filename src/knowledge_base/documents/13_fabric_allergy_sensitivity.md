# 面料过敏与敏感肌指南

## 概述

大约 20% 的人有不同程度的皮肤敏感问题。选对面料可以大幅减少瘙痒、红疹、湿疹等不适。

---

## 常见过敏原面料

### 羊毛（最常见的面料过敏原）
- 过敏原来自羊毛纤维表面的鳞片（物理刺激）和羊毛脂（化学刺激）
- 不是过敏体质也可能感觉"扎"
- **替代方案**：美利奴羊毛（纤维更细 < 20 微米）、羊绒（无鳞片结构）、有机棉

### 化纤面料
- 涤纶、锦纶本身不过敏，但**不透气导致汗液滞留**可能诱发皮疹
- 化纤加工残留的化学品（甲醛、染料、柔顺剂）才是真过敏原
- **替代方案**：选择 Oeko-Tex 认证的化纤产品

### 橡胶/弹性纤维
- 乳胶过敏者对弹力带、松紧腰带可能起反应
- **替代方案**：无弹力设计的纯棉/亚麻衣物

### 金属配件（镍过敏）
- 拉链、纽扣、铆钉中的镍是最常见金属过敏原
- 约 15% 的女性和 2% 的男性对镍过敏
- **预防**：接触皮肤面的金属配件涂一层透明指甲油隔离

## 推荐面料（敏感肌友好）

| 面料 | 推荐指数 | 原因 |
|------|----------|------|
| 有机棉 | ★★★★★ | 无农药残留、无化学加工、透气 |
| 竹纤维 | ★★★★ | 天然抗菌、极其柔软 |
| 莫代尔 | ★★★★ | 比棉更光滑、纤维表面不会摩擦皮肤 |
| 真丝 | ★★★★ | 天然蛋白质纤维、低致敏 |
| 天丝 (Tencel) | ★★★★ | 闭环生产工艺、无残留化学物质 |
| 亚麻 | ★★★ | 透气但纤维偏粗、可能轻微扎 |

## 婴幼儿服装面料

**婴幼儿皮肤角质层只有成人的 1/3 厚度**，对面料的敏感度远高于成人。

### 选购原则
1. **A 类标准**：中国童装安全技术类别中，婴幼儿（<3 岁）必须符合 A 类（甲醛含量 ≤20mg/kg）
2. **浅色优于深色**：染料越少越好
3. **无荧光增白剂**：看标签是否标注
4. **标签外置**：避免标签摩擦婴儿后颈
5. **纯棉首选**：新生儿只穿纯棉
6. **新衣必洗**：去除生产过程中的化学残留

### 推荐与避雷

| 推荐 | 避雷 |
|------|------|
| 有机棉、竹纤维、莫代尔 | 化纤混纺（不透气）、粗羊毛（刺痒） |
| 无骨缝制工艺（减少摩擦） | 过多蕾丝、亮片等装饰 |
| 可按扣全开的设计（方便穿脱） | 套头款（新生儿颈部弱） |

## 新衣服为什么要洗？

新衣服上可能残留：
1. **甲醛**：用于防皱、防缩整理的助剂。贴身衣物 = 甲醛过敏高风险
2. **染料固色剂**：未完全固定的染料会渗到皮肤
3. **防霉剂**：运输仓储过程中的化学保护剂
4. **试衣残留**：你不知道之前有多少人试过这件衣服

**一句话**：贴身穿的、给小孩穿的，必须洗了再穿。外套可以偷懒。

## 常见问题

**Q: 穿了某件衣服就痒，怎么判断是对什么过敏？**
A: 排除法：洗完再穿还痒 → 可能是面料本身（纯羊毛最常见）。洗后不痒 → 是化学残留，再洗一次就好了。只在金属接触处痒 → 镍过敏。

**Q: 皮肤敏感的人应该穿什么？**
A: 有机纯棉或莫代尔是首选。避开粗羊毛、化纤和过多染料。白色/浅色的安全性远高于深色。试新品牌时先买一件穿几天再加购。

## 关键要点

- 羊毛扎人不一定是过敏，也可能是物理刺激
- 婴幼儿只穿 A 类纯棉，这是国家标准也是最低底线
- 新衣服洗了再穿，尤其贴身的
- 皮肤敏感的人从有机棉/莫代尔开始，逐步尝试其他面料
- 衣服的痒，大多是化学残留——再洗一次试试
