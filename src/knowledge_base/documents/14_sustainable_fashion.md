# 可持续时尚与环保购物

## 概述

时尚产业是全球第二大污染源（仅次于石油）。每一次购物选择都在投票——投给快时尚的污染模式，还是可持续的生产方式。

---

## 什么是可持续时尚？

可持续时尚不仅是"环保面料"，而是一个涵盖面料、生产、消费和废弃的全链条理念：

1. **环保面料**：有机种植 + 闭环生产 + 可降解
2. **公平生产**：工人权益、合理工资、安全环境
3. **理性消费**：少买精买、延长衣物寿命
4. **循环利用**：二手、改造、回收

---

## 环保面料图谱

| 面料 | 环保评分 | 说明 |
|------|----------|------|
| 有机棉 | ★★★★★ | 无农药种植、节水、可降解 |
| 天丝 (Tencel/Lyocell) | ★★★★★ | 木浆来源、闭环溶剂回收率 99%+ |
| 大麻 (Hemp) | ★★★★★ | 不需要农药、节水、纤维强韧 |
| 再生涤纶 (rPET) | ★★★★ | 回收塑料瓶制成、但洗涤释放微塑料 |
| 回收棉 | ★★★★ | 边角料再利用 |
| 羊毛 | ★★★ | 可再生但不一定人道（看 mulesing-free 认证） |
| 传统棉花 | ★★ | 极度耗水、全球农药使用量最大的作物之一 |
| 传统涤纶 | ★ | 石油衍生、不可降解、微塑料污染 |

### 认证标志
- **GOTS**（全球有机纺织品标准）：有机纤维 70%+
- **Oeko-Tex Standard 100**：无有害化学物质（但不是有机认证）
- **BCI**（良好棉花倡议）：可持续棉花种植
- **FSC**（森林管理委员会）：木浆来源合法且可持续

## 胶囊衣橱概念

**定义**：用最少的单品（通常 30-40 件，含鞋包），搭配出最多的穿法。

**构建步骤**：
1. 清空衣橱，只留真正穿的（80% 没穿过的衣服说明你不需要它们）
2. 基础款优先：白 T、黑裤、牛仔裤、风衣、小黑裙
3. 配色统一：选定 2-3 个主色 + 1-2 个点缀色
4. 一衣多穿：买之前想 3 种搭配方式
5. 每季更新不超过 3 件

**益处**：省时间（减少决策疲劳）、省钱、省空间、环保

## 二手选购指南

### 值得买二手的
| 品类 | 理由 |
|------|------|
| 奢侈包 | 经典款二手价格友好，且皮具越用越有味道 |
| 大衣/风衣 | 经典款永不过时 |
| 珠宝/手表 | 二手价优，品质不减 |
| 牛仔裤 | 已经被"养"过了，软硬适中 |

### 不建议买二手的
- 贴身内衣（卫生问题）
- 鞋子（前主人的脚型已定型在鞋内）
- 快时尚品牌（新品也不贵，二手更是贬值严重）

## 如何延长衣物寿命

一件衣服多穿 9 个月 = 减少 20-30% 的碳足迹和水足迹。

1. 看水洗标、按说明洗
2. 减少洗涤频率（牛仔裤穿 5-10 次再洗）
3. 掌握基本的修补技能（缝纽扣、补小洞）
4. 换季正确收纳（洗净 → 干燥 → 防蛀 → 存放）

## 常见问题

**Q: 可持续品牌的衣服为什么贵？**
A: 环保面料生产成本高 + 公平工资 + 认证费用。但一件有机棉 T 恤穿 3 年 vs 一件快时尚 T 恤穿 3 个月，年均成本其实更低。

**Q: 扔掉的旧衣服去哪了？**
A: 大部分（约 85%）被填埋或焚烧。只有不到 1% 被回收再制成新衣服。这也是为什么"少买"比"回收"更重要。

## 关键要点

- 最环保的衣服 = 你已经拥有的那件
- 买之前问自己：我会穿它超过 30 次吗？——不够 30 次就别买
- 少买、买好、穿久——可持续时尚的核心就六个字
- 天丝和有机棉是当前最成熟的两大环保面料
