# 网购服装避坑指南

## 概述

网购服装最大的痛点：看着模特图美到窒息，收到实物想退货。掌握以下技巧，可以大幅降低翻车率。

---

## 看懂商品详情页

### 关键信息优先级
1. **面料成分** > 所有。聚酯纤维写的"重磅真丝手感" = 还是聚酯纤维
2. **尺码数据** > 模特上身效果。模特 170/45kg 穿 S，你 160/55kg 要穿 L
3. **买家秀** > 模特图。看和你身材相似的买家拍的图
4. **问答区** > 评价区。有针对性问题的答案

### 标题套路的真相

| 标题写入 | 实际含义 |
|----------|----------|
| "重磅" | 面料偏厚，但不代表品质好 |
| "高端定制" | 大概率是通货，不是定制 |
| "韩版" | 大概率不是韩国产，是"偏小"的委婉说法 |
| "网红同款" | 和网红穿的可能只是"远看同款" |
| "真丝手感" / "羊绒手感" | 不是真丝！不是羊绒！只是手感模仿 |
| "含羊毛" | 含多少？1% 也算含 |

---

## 看图识货

### 警惕这些信号
- 模特全是坐着的（遮住版型问题）
- 没有无修图的平铺图或背面图
- 细节图只拍好看的一角
- 模特姿势遮住了关键部位（交叉手臂遮肚子、侧身挡腰线）
- 全部是外模图（亚洲人身材差别大）

### 好卖家的标志
- 有平铺测量图（衣服摊平，旁边放尺子）
- 有不同身材的模特上身（160/165/170 各一）
- 有面料近拍细节
- 有室内自然光 + 室外光的对比图

---

## 评论区的正确读法

- 看追评（穿过洗过后的评价 > 刚收到的评价）
- 看差评（差评比好评更有信息量——好评可能是刷的，差评一般不是）
- 看"有图/视频"的评价（最有说服力）
- 排除"与预期不符"这类主观评价，关注"掉色""缩水""起球"等客观描述
- 中评往往比好评更可信

---

## 退货策略

1. **买两码退一码**：多花一笔运费，但比收到不合身再换省时间
2. **开箱录视频**：保留证据，防止卖家以"穿过不能退"拒绝
3. **保留吊牌**：吊牌剪了就不能退了
4. **退货运费险**：经常网购的一定要买
5. **7 天无理由**：法律保障，卖家拒退直接平台介入

## 促销避坑

| 套路 | 真相 |
|------|------|
| "原价 999，现价 199" | 大概率这件衣服就值 199 |
| "最后一件" | 库存压力制造紧迫感 |
| "XX 同厂" | 不同生产线、不同面料、不同品控 |
| 满减凑单 | 为了凑单买的通常都是垃圾 |
| 限时秒杀 | 真的好东西不会挂秒杀（除了大促清仓） |

---

## 不同品类的选购要点

| 品类 | 重点看什么 |
|------|-----------|
| 大衣/风衣 | 面料成分、衬里材质、肩宽和袖长 |
| 毛衣/针织 | 纱线成分（羊毛/羊绒/棉/化纤比例）、是否有起球反馈 |
| 牛仔裤 | 棉含量、弹性（氨纶%）、裤长、大腿围数据 |
| 连衣裙 | 胸围、腰围、面料是否有弹性、有无内衬 |
| 鞋子 | 跟高、鞋宽、是否有磨脚评价、买大半码还是小半码 |
| T 恤 | 克重（180-220g 最佳：不薄不透也不厚重）、领口加固工艺 |
| 羽绒服 | 充绒量(g)、含绒量(%)、蓬松度(FP)、面料防钻绒处理 |

## 常见问题

**Q: 为什么同一件衣服，不同颜色穿起来感觉不一样？**
A: 黑色染料会使面料变硬/收缩（化学作用），所以同款不同色尺码手感可能不同。看买家对不同颜色的评价。

**Q: "原单""尾单"靠谱吗？**
A: 99% 是假的。真正的原单（代工厂多做的几件）在行业内会被品牌方严格管控，几乎不会流到普通消费者手里。市面上的"原单"基本是高仿的代名词。

**Q: 直播间买衣服风险大吗？**
A: 主播上身 + 打光 + 滤镜 = 实物差距可能很大。一些直播间不支持 7 天无理由。除非退货方便，否则先买一件试水再考虑回购。

## 关键要点

- 面料成分水洗标 = 服装的身份证，比标题重要 100 倍
- 模特图是卖家想让你看到的，买家秀是你实际会收到的
- 买两码退一码是网购常态，不要觉得不好意思
- 便宜衣服的问题往往在穿 5 次以后才显现（起球、变形、褪色）
- 基础款多花点钱（能穿很久），潮流款买便宜的（穿一季就够了）
